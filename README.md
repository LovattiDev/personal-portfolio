# Personal Portfolio

A modern and responsive personal portfolio website built to showcase my projects, skills, and professional experience as a Software Engineer.


## Purpose

This project was developed to:
- Showcase my real-world projects
- Create a professional online presence
- Practice front-end development and UI/UX design
- Serve as a central hub for recruiters and collaborators


## Features

- **Responsive Design** – Works on desktop and mobile devices
- **Dynamic Contact Form** – Allows users to send messages directly
- **Project Showcase** – Displays selected projects with descriptions and links
- **Modern UI** – Clean and minimal interface


## Technologies

- HTML5
- CSS3
- JavaScript
- Git & GitHub


## Highlight Project

This portfolio includes projects focused on real-world problems, such as maritime safety solutions based on real accident data.


## What I Learned

- Structuring a scalable front-end project
- Creating reusable components
- Handling form validation and integration with backend
- Improving UI/UX design principles
